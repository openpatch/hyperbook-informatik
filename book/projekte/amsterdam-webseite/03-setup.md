# Setup

Zuerst installieren wir VS Code. Das ist ein kostenloser Editor, der dir das Programmieren erleichtert. Du kannst ihn hier herunterladen: [VS Code](https://code.visualstudio.com/).

Außerdem installieren wir die Erweiterung "Live Preview". Das ist eine Erweiterung, die dir ermöglicht, deine Webseite direkt im integrierten Browser zu sehen, während du sie schreibst.
