# Pubquiz Amsterdam

Wir wollen ein Pubquiz auf unserer Webseite einfügen. Das Pubquiz soll Fragen über Amsterdam enthalten. Die Fragen sollen in einer Liste dargestellt werden. Jede Frage soll eine Überschrift und eine Antwort haben. Die Antwort soll erst sichtbar werden, wenn der Benutzer auf die Frage klickt.
