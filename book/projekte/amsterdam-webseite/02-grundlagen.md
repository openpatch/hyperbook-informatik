# Grundlagen

Während des Projekts werden wir HTML, CSS und JavaScript kennenlernen. Damit du diese gut nutzen kannst, verlinke ich dir die Grundlagen, die du für den aktuellen Schritt im Projekt benötigst.

Diese führen dich zu einer Seite dieses Hyperbooks auf der du mit interaktiven Übungen die Grundlagen lernen kannst.

Die Übungen sehen zum Beispiel so aus:

:::webide

```html
<h1>Meine erste Webseite</h1>
<p>Willkommen auf meiner Webseite!</p>
```

```css
html, body {
    background: darkslategray;
    color: darkseagreen;
    padding: 16px;
}

h1 {
    color: darksalmon;
}
```

:::

Diese aktualiisiert sich automatisch, sobald du etwas änderst. So kannst du direkt sehen, wie sich dein Code auf die Webseite auswirkt.

:pen: Ändere doch einfach mal "Meine erste Webseite" in "Die beste Webseite" und schau dir an, was passiert.
