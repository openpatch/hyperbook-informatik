# Installation von OpenSCAD

Du kannst OpenSCAD von der offiziellen Webseite herunterladen: [https://www.openscad.org/downloads.html](https://www.openscad.org/downloads.html).

In diesem Hyperbook kannst du OpenSCAD auch direkt im Browser
verwenden, ohne das Programm installiert zu haben. Um die
Funktionen von OpenSCAD zu lernen reicht das vollkommen aus,
für komplexere Projekte ist die Installation auf deinem
Computer aber empfehlenswert, da sie performanter funktioniert.

Auf der nächsten Seite findest du eine Einführung in die Bedienung der lokal installierten und der browserversion von OpenSCAD.
