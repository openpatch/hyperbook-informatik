# Über OpenSCAD

OpenSCAD ist eine Software, die es ermöglicht, 3D-Modelle durch Programmieren zu erstellen. Im Gegensatz zu anderen 3D-Modellierungsprogrammen, die auf grafische Benutzeroberflächen setzen, basiert OpenSCAD auf einer Skriptsprache, die es den Benutzern erlaubt, präzise und wiederholbare Modelle zu erstellen.

https://openscad.org

OpenSCAD ist besonders beliebt in der Maker-Community und bei
3D-Druck-Enthusiasten, da es eine einfache Möglichkeit bietet, komplexe
geometrische Formen zu erstellen und zu modifizieren. Es unterstützt auch die
Erstellung von parametrischen Modellen, was bedeutet, dass Benutzer Variablen
verwenden können, um die Größe und Form ihrer Modelle anzupassen.

Dabei wird textbasiert gearbeitet, was es ermöglicht, Modelle
durch das Schreiben von Code zu erstellen. Dies kann besonders
nützlich sein, wenn man komplexe oder wiederholbare Designs
erstellen möchte, da man einfach die Parameter ändern kann, um
verschiedene Versionen eines Modells zu generieren. OpenSCAD
bietet auch eine Vorschau-Funktion, mit der Benutzer ihre
Modelle in Echtzeit sehen können, während sie den Code
schreiben.

OpenSCAD verwendet eine ähnliche Syntax wie die Programmiersprache C oder Java
und eignet sich daher auch gut als erster Einstieg in die Programmierung.
