# Projekte

In den bisherigen Kapiteln waren die Aufgaben klein und die Lösung meist absehbar. In einem **Projekt** ist das anders: Es gibt kein festes Ziel, das man abhaken kann, sondern ein Vorhaben, das ihr selbst gestaltet.

Die drei Projekte laufen jeweils in mehreren **Phasen** ab. Ihr arbeitet allein oder zu zweit und gebt euch zwischendurch gegenseitig Rückmeldung.

| Projekt | Passt zu Kapitel | Schwerpunkt |
| --- | --- | --- |
| Zahlenraten | 2 | Variablen, Schleifen, Verzweigungen |
| Unsere kleine Stadt | 4 | Funktionen, Zusammenarbeit |
| Memory | 5 | Listen, bestehenden Code verstehen |
