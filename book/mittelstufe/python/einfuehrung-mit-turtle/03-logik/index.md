# Logik

Bisher hatten deine Bedingungen immer nur eine einzige Frage: *„Ist die Zahl gerade?"* oder *„Ist genug Geld auf dem Konto?"*

Im Alltag hängen Entscheidungen aber meist von **mehreren Dingen gleichzeitig** ab: Eis gibt es nur, wenn die Sonne scheint **und** genug Geld da ist.

In diesem Kapitel lernst du, wie man Bedingungen miteinander verknüpft.
